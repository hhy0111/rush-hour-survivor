# ADS_SETUP

## Current Recommendation

- SDK family: AdMob
- MVP ad placements:
  - post-run placement after a run ends
  - rewarded revive entry point only if the economy stays fair

## Do Not Do

- Do not block the first run with ads
- Do not interrupt a session mid-chaos for a forced ad
- Do not hardcode live IDs

## Connection Plan

- Human-entered values: real App ID and Unit IDs
- Code-connected values: placeholder env keys in `.env.sample`
- Console/store values: Play Console ad disclosure settings
