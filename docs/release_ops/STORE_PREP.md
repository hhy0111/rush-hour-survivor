## Google Play Full Description

퇴근길은 오늘도 전쟁이다.

Rush Hour Survivor는 지하철, 버스, 거리에서 밀려드는 인파를 피해 살아남는 터치 기반 로그라이크 생존 액션 게임입니다. 단순히 오래 버티는 것만으로 끝나지 않습니다. 체력과 스트레스를 동시에 관리하면서 점점 더 혼잡해지는 퇴근길을 돌파해야 합니다.

플레이어는 직장인이 되어 복잡한 도시의 퇴근 러시를 버텨냅니다. 커피로 체력을 회복하고, 이어폰으로 스트레스를 낮추고, 택시 호출로 위기에서 벗어나세요. 시간이 지날수록 지각 위험, 상사 전화, 폭우 같은 변수들이 겹쳐지며 생존 난이도가 빠르게 올라갑니다.

게임 특징
- 5분 내외로 압축한 몰입형 생존 플레이
- 터치 중심의 직관적인 조작
- 체력과 스트레스를 함께 관리하는 이중 생존 구조
- 커피, 이어폰, 택시 호출 등 테마형 아이템 활용
- 지하철, 버스, 거리의 퇴근 혼잡을 모티브로 한 긴장감 있는 진행
- 로컬 저장 기반의 최고 점수 기록
- 오프라인 환경에서도 핵심 플레이 가능

이런 플레이를 좋아한다면 추천합니다
- 짧지만 집중도 높은 생존 액션을 좋아하는 플레이어
- 한 판 한 판 다시 도전하는 로그라이크 구조를 선호하는 플레이어
- 현대 도시 생활의 공감 가는 테마를 색다른 게임으로 즐기고 싶은 플레이어

Rush Hour Survivor에서 오늘의 퇴근길을 끝까지 버텨보세요.
