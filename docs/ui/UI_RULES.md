# UI_RULES

## Layout

- Portrait-first composition around a `390 x 844` base viewport
- HUD must not cover the player origin or pickup spawn hotspots
- Avoid more than four simultaneous numeric HUD values

## Readability

- High-contrast text over tinted panels
- Strong distinction between damage-related red tones and stress-related amber tones
- Use familiar casual-game visual patterns, not sci-fi dashboards

## Interaction

- Start and restart buttons are the only primary CTAs in the current MVP
- Do not hide essential actions behind gesture-only discovery
- Touch behavior must remain understandable without a tutorial wall of text
