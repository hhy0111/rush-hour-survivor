# QA_SEVERITY_RULES

## Blocker

- Prevents progress, testing, or release
- Breaks start/play/fail/restart loop
- Corrupts save or causes unrecoverable stuck state

## Major

- Harms the core experience or monetization path
- Makes touch interaction confusing
- Produces strong commercial-quality or AI-artifact concerns

## Minor

- Worth fixing before release but does not block current gate by itself
- Copy, spacing, non-critical visual polish, or low-risk edge behavior
